# 环境安装
## 安装node环境

* 1.安装node.js

```text
node.js的安装地址：https://nodejs.org/en/download/
```

* 2.安装yarn

```shell
npm install -g yarn --registry=https://registry.npm.taobao.org
```

* 如果遇到权限问题，在命令前边加上sudo 

## 安装开发工具

* WebStorm
